If cause = condition 
\
Main cause = Result

#### Five Types:
- "0 Conditional"
- "1<sup>st</sup> Conditional" 
- "2<sup>nd</sup> Conditional"
- "3<sup>rd</sup> Conditional"
- "Mixed Conditional


## 0 Conditional

## 1<sup>st</sup> Conditional
### `If + present simple, + subject + will + base form`

Si usa per esprimere ***ipotesi possibili***

## 2<sup>nd</sup> Conditional
### `If + past simple, + subject + would + base form`

Si usa per esprimere ***ipotesi impossibili o improbabili***

**REMEMBER: Would can be used with `may(+formal)/could/might`**

## 3<sup>rd</sup> Conditional
### `If + past perfect, + subject + would have + past participle`

Si usa per esprimere ***ipotetoche*** situazioni nel ***passato*** o per un ***rimpianto***

**REMEMBER: Would can be used with `may(+formal)/could/might`**
